# AI-Dealership-Service-and-Repair-Scheduling-System
LLM-Powered AI Dealership Service Management System 🚗🤖

An intelligent dealership automation platform developed using Python, Streamlit, Machine Learning, and Google Gemini AI. The system provides AI-based vehicle issue prediction, LLM-powered maintenance assistance, appointment scheduling, analytics dashboard, and dealership workflow automation.

🔹 Features:
• Vehicle Issue Prediction using TF-IDF + RandomForest
• Google Gemini LLM Integration
• AI Maintenance Assistant
• Service Appointment Management
• Analytics & Reporting Dashboard
• SQLite Database Integration
• Intelligent Customer Support System

🔹 Technologies Used:
Python • Streamlit • scikit-learn • SQLite • Google Gemini AI • Pandas • Machine Learning • LLM Integration

This project demonstrates practical implementation of Artificial Intelligence, Machine Learning, and Large Language Models in real-world dealership service management and intelligent workflow automation.
